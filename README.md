# kubeflow-samples

Kubeflow & Kfserving Codes.


## Auto Completion

* Bash

```bash
sudo apt install -y bash-completion
## Bash
echo 'source <(kubectl completion bash)' >>~/.bashrc
## Zsh
source <(kubectl completion zsh)
```

```bash
sudo yum install -y bash-completion
## Bash
echo 'source <(kubectl completion bash)' >>~/.bashrc
## Zsh
source <(kubectl completion zsh)
```


* Zsh(oh-my-zsh)

```zsh
plugins=(git ... kubectl)
```
